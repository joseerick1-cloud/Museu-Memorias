# Documentação de arquivos do Projeto Museu Memórias

Este documento descreve cada arquivo do projeto e mostra onde encontrar as explicações completas de HTML e CSS em `docs/`.

## 1. Objetivo do projeto

O site simula um museu digital com páginas que mostram exposições, acervo e histórias orais. A página principal apresenta o museu, e as demais páginas exibem conteúdo dividido em categorias de navegação.

## 2. Arquivos do projeto

- `inicio.html` — página inicial
- `exposicao.html` — página da exposição em cartaz
- `acervo.html` — página geral do acervo
- `fotografias.html` — página de fotografias históricas
- `objetos.html` — página de objetos do acervo
- `documentos.html` — página de documentos históricos
- `historia-oral.html` — página de depoimentos e entrevistas
- `styles.css` — arquivo de estilo principal
- `README.md` — documentação geral do projeto

## 3. Onde encontrar a explicação completa

- `docs/inicio.md` — explica `inicio.html` e as classes usadas no banner, botões e rodapé.
- `docs/exposicao.md` — explica `exposicao.html`, incluindo hero, galeria, linha do tempo, depoimento e formulário.
- `docs/acervo.md` — explica `acervo.html`, a navegação por abas e o layout de galeria.
- `docs/fotografias.md` — explica `fotografias.html`, os cards de imagem e metadados.
- `docs/objetos.md` — explica `objetos.html`, os cards de objetos físicos e a sidebar.
- `docs/documentos.md` — explica `documentos.html`, os registros escritos e os placeholders de imagem.
- `docs/historia-oral.md` — explica `historia-oral.html`, as entrevistas e as imagens de depoimento.
- `docs/styles.md` — explica `styles.css`, incluindo variáveis, header, hero, grid, placeholders, botões e responsividade.

## 4. Como usar esta documentação

1. Abra o arquivo HTML correspondente.
2. Consulte o `docs/*.md` para entender as classes CSS e a estrutura de cada página.
3. Compare o HTML com `styles.css` para ver como o layout e as imagens são construídos.

## 5. Observação

Os arquivos de documentação em `docs/` não alteram o site. Eles servem para explicar o funcionamento do HTML e do CSS para apresentação e revisão no GitHub.
